# After the Bell

A static digital counterstory website about poverty, work, and education. It uses textual storytelling, real photos, CSS infographics, scroll-based spatial layout, and sourced data cards.

## Files

- `index.html` contains the story and source links.
- `styles.css` creates the visual layout and scrollytelling scenes.
- `script.js` handles scroll progress and scene changes.
- `assets/` contains the local images used by the website.

## Free hosting options

### GitHub Pages

1. Create a public GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and `README.md`.
3. Go to `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/root`, then save.
6. GitHub will give you a free URL like `https://yourusername.github.io/repository-name/`.

### Replit

1. Create a new Replit project.
2. Choose `HTML, CSS, JS`.
3. Upload or paste the three website files.
4. Press `Run`.
5. Use Replit's preview/share URL for submitting the assignment.

## Research note

Maya is a fictional composite character. The story is based on patterns from student-voice reporting and statistics from the NCES `Report on the Condition of Education 2024`.
